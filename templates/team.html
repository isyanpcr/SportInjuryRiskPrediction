Tujuan Proyek
Membangun model ANN untuk prediksi risiko cedera atlet berbasis data biometrik dan latihan.
Mengolah dataset SIRP-600 melalui preprocessing, encoding, dan normalisasi fitur.
Mengevaluasi performa model (akurasi, confusion matrix, grafik loss & accuracy).
Menghasilkan sistem klasifikasi risiko cedera: Low Risk atau High Risk.
Dataset — SIRP-600
Sumber Kaggle · SIRP-600 ↗
Jumlah Sampel 600 Atlet
Komposisi Gender 335 Wanita · 265 Pria

Tahapan Proyek (Pipeline)
1
Data Preprocessing
Pembersihan data, handling missing values, dan verifikasi tipe data dari SIRP-600.
2
Feature Encoding
Encoding fitur kategorikal (Gender: 0/1) agar dapat diproses oleh model ANN.
3
Normalisasi Data
StandardScaler diterapkan pada seluruh fitur numerik untuk mempercepat konvergensi ANN.
4
Pembangunan Model ANN
Arsitektur ANN dengan layer Dense, fungsi aktivasi ReLU, dan output Sigmoid biner.
5
Training & Testing
Model dilatih pada data training dan dievaluasi dengan data uji terpisah.
6
Hasil & Rekomendasi
Threshold ≥ 0.5 → High Risk. Output disajikan bersama rekomendasi preventif berbasis data.
